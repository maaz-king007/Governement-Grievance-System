System Capabilities

🏢

Smart Department Routing

Automatically classifies grievances into Banking, Telecom, Electricity, Railways, E-Commerce, Healthcare & Government departments using LinearSVC

💬

Sentiment Analysis

Detects positive, neutral, and negative sentiment in code-mixed Hinglish text using Logistic Regression with TF-IDF features

🚨

Urgency Detection

Hybrid rule-based + ML urgency scoring ensures emergency complaints receive immediate priority escalation

🚫

Hate Speech Detection

Identifies abusive / hateful language in complaints using the Code-Mixed Hateful Hinglish dataset patterns

🗣️

Hinglish NLP

Native support for Roman Hinglish, code-mixed text, and Indic transliteration with custom preprocessing pipeline

⚡

Instant Auto-Response

Context-aware response generation based on department + urgency combination with actionable next steps for users
